# URL Shortener

## Dependencies

- Go version 1.21
- cleanenv by ilyakaznacheev
- chi version 4

## Setup

### Install Go

`sudo make setup`

### Upgrade Go

`sudo make install-go`

### Build project

`sudo make build`

### Build and run project

`sudo make run`
